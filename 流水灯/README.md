# 流水灯项目

## 1. 项目简介
- 项目名称：流水灯
- 项目描述：基于STM32F10x单片机的LED流水灯演示程序，带有蜂鸣器音效

## 2. 硬件说明
- 单片机型号：STM32F10x系列
- LED引脚：PA0、PA1、PA2、PA3
- 蜂鸣器引脚：PB12
- GPIO配置：推挽输出模式，50MHz速度

## 3. 软件说明
- 开发环境：STM32标准外设库
- 主要功能函数：
  - `delay_ms()`：毫秒级延时函数
  - `beep()`：蜂鸣器控制函数
  - `main()`：主程序

## 4. 功能说明
- 4个LED依次点亮
- 每个LED点亮时伴随蜂鸣器音效
- 音效频率递增（20、22、24、26）
- 每个LED点亮时间约300ms（5×60ms）

## 5. 编译与烧录
- 编译工具：Keil MDK或其他ARM编译器
- 烧录方式：ST-Link或J-Link调试器

## 6. 注意事项
- 确保硬件连接正确
- 根据实际晶振频率调整延时参数
- 注意GPIO引脚复用冲突
