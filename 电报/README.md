# 电报项目

## 1. 项目简介
- 项目名称：摩尔斯电报
- 项目描述：基于STM32F10x单片机的摩尔斯电码发送程序，通过蜂鸣器和LED输出

## 2. 硬件说明
- 单片机型号：STM32F10x系列
- 蜂鸣器引脚：PB12（低电平有效）
- LED引脚：PA0（低电平有效）
- GPIO配置：推挽输出模式，50MHz速度

## 3. 软件说明
- 开发环境：STM32标准外设库
- 主要功能函数：
  - `Beep_Init()`：蜂鸣器和LED初始化
  - `delay_ms()`：毫秒级延时函数
  - `Beep_On()` / `Beep_Off()`：蜂鸣器和LED控制
  - `play_dot()`：播放点（短音）
  - `play_dash()`：播放划（长音）
  - `play_char()`：播放单个字符
  - `play_string()`：播放字符串
  - `main()`：主程序

## 4. 功能说明
- 支持A~Z字母的摩尔斯电码编码
- 通过蜂鸣器和LED同步输出摩尔斯电码
- 摩尔斯电码时间参数：
  - 点：120ms
  - 划：360ms
  - 间隔：120ms
  - 字符间隔：360ms
  - 单词间隔：720ms
- 默认发送内容：SOS

## 5. 编译与烧录
- 编译工具：Keil MDK或其他ARM编译器
- 烧录方式：ST-Link或J-Link调试器

## 6. 注意事项
- 确保硬件连接正确
- 根据实际晶振频率调整延时参数
- 可通过修改CONTENT宏定义更改发送内容
- 注意GPIO引脚复用冲突
